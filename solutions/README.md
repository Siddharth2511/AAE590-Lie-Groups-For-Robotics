# Problem-Set Solutions

This folder contains my solution artifacts for **Lie Group Methods for Estimation and Control**. It is intentionally limited to work products I created: notebooks, code, generated plots, and small supporting models.

Course problem statement PDFs, prompt handouts, and assignment sheets are not included.

## Contents

| Folder | Description |
|---|---|
| [ps04](ps04) | `SE(2)` and unicycle solution notebooks, a Modelica unicycle model, a RUMOCA comparison script, and generated overlay results. |

## Notes

- Notebook outputs are retained where they help show the computation and results.
- Some optional scripts depend on external modeling tools such as RUMOCA.
- The repository keeps solution work separate from course handouts so the GitHub page can remain useful without redistributing protected course material.
