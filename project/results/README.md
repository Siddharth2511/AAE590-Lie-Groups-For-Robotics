# Results

This folder contains generated summaries from the particle-filter project.

## Reports

- `report.md` - main simulation and filter-comparison writeup
- `hardware_calibration_ablation.md` - hardware-calibration hypothesis study

## Data

- `data/summary.csv` - scenario-level filter metrics
- `data/pf_particle_sweep.csv` - PF particle-count sweep
- `data/ambiguous_candidate_summary.csv` - ambiguity/false-detection study
- `data/hardware_calibration_ablation.csv` - calibration-error ablation metrics

## Figures

The `figures/` directory contains the plots and selected GIFs used in the top-level README and project reports.
